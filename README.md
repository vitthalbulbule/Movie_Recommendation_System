# Movie_Recommendation_System



This project recommends movies using content-based filtering.  
Built using Python, Pandas, Numpy, CountVectorizer, and Cosine Similarity.

# How it works
1. Data cleaning  
2. Tag generation  
3. Vectorization  
4. Similarity calculation  
5. Recommend top 5 similar movies

# Technologies
- Python
- Jupyter Notebook
- Pandas
- NLTK
- Scikit-learn
